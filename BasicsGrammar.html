MarkDownPad2基本语法

            

            

            

                


    
一、换行和空格  

(1)换行  

行尾加两个空格  

(2)空格  

&nbsp;  




二、标题  

在#后跟个空格再写文字，一个#是一级标题，两个#是二级标题，以此类推，支持六级标题。  

示例：

# 一级标题

## 二级标题

### 三级标题



显示效果：  







三、字体  

(1)加粗  

要加粗的文字左右分别用两个*号包起来。



示例：  

**这是要加粗的文字**



显示效果：  





(2)斜体  

要倾斜的文字左右分别用一个*号包起来。



示例：  

*这是要倾斜的文字*



显示效果：  



(3)斜体加粗  

要斜体加粗的文字左右分别用三个*包起来。 



示例：  

***这是斜体加粗的文字***  



显示效果：







四、引用  

在引用的文字前加>即可，引用也可以嵌套，如加两个>>三个>>>



示例：

>引用的内容

>>引用的内容1

>>>引用的内容2  



显示效果：  




 


 


五、分割线  

三个或三个以上的*。 



示例：

快要分隔了  

***



显示效果：  




 



六、超链接  

[超链接名](超链接地址 "超链接描述")  



示例:

[Google](https://www.google.com.hk/ "谷歌")  



显示效果：  




 



七、列表  

(1)无序列表  

*加空格。 



示例：

* 列表内容1  

* 列表内容2  

* 列表内容3



显示效果：



(2)有序列表  

数字加点加空格。  



示例：

1. 有序列表1  

2. 有序列表2  

3. 有序列表3  



显示效果：



(3)列表嵌套  

在上一级和下一级之间敲三个空格即可。 



示例： 

1. 有序列表1

   * 无序列表1

   * 无序列表2

2. 有序列表2

   * 无序列表1

   * 无序列表2



显示效果：  







八、代码  

(1)单行代码  

代码之间分别用一个反引号包起来。  



示例：  

`System.out.println("Hello World");`



显示效果：  





(2)多行代码  

代码之间分别用三个反引号包起来，且两边的反引号单独占一行。



示例：  

```

while(true)  

{  

    System.out.println("Hello World");  

}

```



显示效果：
